
# tmux 命令
```bash
tmux list-panes
tmux capture-pane -p -t %2 2>/dev/null | tail -50
tmux kill-pane -t %2
```

# tmux 命令说明

## tmux capture-pane

这是一个 tmux（终端复用器）命令，用于捕获指定窗格的输出内容。

  命令分解：

  ┌───────────────────┬───────────────────────────────────────────────────────┐
  │       部分        │                         含义                          │
  ├───────────────────┼───────────────────────────────────────────────────────┤
  │ tmux capture-pane │ 捕获指定窗格的文本内容                                │
  ├───────────────────┼───────────────────────────────────────────────────────┤
  │ -p                │ 将捕获的内容输出到 stdout（而不是粘贴到 tmux 缓冲区） │
  ├───────────────────┼───────────────────────────────────────────────────────┤
  │ -t %2             │ 目标窗格，%2 表示第二个窗格（窗格编号从 0 开始）      │
  ├───────────────────┼───────────────────────────────────────────────────────┤
  │ 2>/dev/null       │ 忽略错误输出（如窗格不存在时的报错）                  │
  ├───────────────────┼───────────────────────────────────────────────────────┤
  │ tail -50          │ 只显示最后 50 行                                      │
  └───────────────────┴───────────────────────────────────────────────────────┘
